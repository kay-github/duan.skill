# Round 2 Topic Blocks

- Scope: second-pass grouped extraction built on the new source families

## T11. 集中不是鲁莽，而是“少数真懂 + 不借钱”
- Core view: 他接受高集中，但前提非常苛刻，不是鼓励拍脑袋梭哈。
- Evidence:
  - `cleaned-markdown/round-2/concentration-excerpts.md`: 真正投资过的公司最多五六家，持有一般三家左右。
  - 同文件：很长时间里 80% 都集中在一只股票上。
  - 同文件：不借钱、不做空、不做不懂的东西。
- Extraction summary: 他对集中并不保守，但对“为什么能集中”极其保守。

## T12. 卖出逻辑是“重估 + 理由变化”，不是“赚了就跑”
- Core view: 卖出不是因为成本价、浮盈或短期市场，而是因为价值判断和理由变化。
- Evidence:
  - `cleaned-markdown/round-2/concentration-excerpts.md`: 卖股票和成本无关。
  - 同文件：到了基本目标后要重新评估。
  - 同文件：买的理由消失或负面增加到不可接受时才离场。
- Extraction summary: 这让他的语言天然反对“止盈止损公式化”。

## T13. `margin of safety` 更像“理解上的安全”
- Core view: 安全边际不只是便宜，而是“我真的知道自己在干什么”。
- Evidence:
  - `cleaned-markdown/round-2/concentration-excerpts.md`: `margin of safety` 应该指能力圈而不仅仅是价格。
  - round 1 多处“不懂不做”。
- Extraction summary: 后续 skill 写投资判断时，应避免把安全边际窄化成折价率。

## T14. 文化不是氛围，而是组织的对错系统
- Core view: `Mission / Vision / Core Values` 不是装饰，而是组织对“该做什么 / 不该做什么”的明确定义。
- Evidence:
  - `cleaned-markdown/round-2/stanford-49-excerpts.md`: 企业文化就是 M/V/CV。
  - round 1 已有“文化能管到制度管不到的地方”。
- Extraction summary: 其对文化的理解非常操作化，不是文学化。

## T15. 人的选择优先看价值观匹配
- Core view: “合适”比“合格”更难，也更重要。
- Evidence:
  - `cleaned-markdown/round-2/stanford-49-excerpts.md`: 价值观不 match 的人坚决不要。
  - 同文件：合格但不合适的人往往制造麻烦。
  - 同文件：找到同道中人，淘汰不合适的人。
- Extraction summary: 他看组织问题，首先想到的是选人和筛人，而不是流程修修补补。

## T16. 创始人要守角色边界，不抢 CEO 功劳
- Core view: 真正理解组织的人，不会通过持续露面制造“还是我在管”的幻觉。
- Evidence:
  - `cleaned-markdown/round-2/stanford-49-excerpts.md`: 不抢 CEO 的成就感。
  - round 1 已有“企业文化比单个产品更重要”。
- Extraction summary: 其人格里有明显的“退后一步让系统运转”的倾向。

## T17. 家庭与友谊是其非投资人格的重要组成
- Core view: 他不是纯投资机器；家庭安全感、友谊和少数深关系在其表达中权重很高。
- Evidence:
  - `cleaned-markdown/round-2/stanford-49-excerpts.md`: 家人、亲情、友情最重要。
  - 同文件：给孩子安全感、高质量陪伴。
  - 同文件：能有一打好朋友就足够。
- Extraction summary: future skill 若只会谈股票，不谈做人做事，就会失真。

## T18. “我不懂”是成熟判断的一部分
- Core view: 他并不追求面面俱到，而是坦然承认不懂，把这当作决策质量的一部分。
- Evidence:
  - `cleaned-markdown/round-2/xueqiu-investment-logic-notes.md`: “我不懂”“看不明白”“没研究过”高频。
  - round 1、round 2 多处直接说不懂区块链、银行、前沿等。
- Extraction summary: future skill 必须保留“承认不知道”的能力，否则不像。
