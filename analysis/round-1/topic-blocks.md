# Round 1 Topic Blocks

- Scope: first grouped extraction by recurring topic rather than by single Q/A
- Evidence basis: current cleaned round-1 excerpts plus the Stanford HTML source

## T1. `Stop Doing List` 是总纲，不是单条技巧
- Core view: 他把长期成功理解为“先去掉错的事情”，而不是“快速找到万能招式”。
- Evidence:
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: 不做错的事更重要，确认错了就立刻停。
  - `cleaned-markdown/round-1/investment-logic-excerpts.md`: 秘诀不是做了什么，而是不做什么。
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 好的公司都有长长的 `Stop doing list`。
- Extraction summary: 这是理解段永平方法论的入口词，后续 skill 应优先从“排除什么”来回答，而不是先给操作清单。

## T2. 能力圈的重点是边界感，不是扩张欲
- Core view: 他更警惕“误以为自己懂”，而不是鼓励“尽快扩大边界”。
- Evidence:
  - `cleaned-markdown/round-1/investment-logic-excerpts.md`: 大小不重要，知道边界在哪儿才重要。
  - 同文件：主动待在能力圈内，比被市场打回来要好。
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: 能力圈是诚实对自己，知之为知之，不知为不知。
- Extraction summary: 其“能力圈”概念带有强烈的自知之明和反自欺成分。

## T3. 投资本质是买生意、买未来现金流
- Core view: 股票只是壳，真正要理解的是生意模式和未来自由现金流。
- Evidence:
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 买股票就是买公司。
  - `cleaned-markdown/round-1/investment-logic-excerpts.md`: 投资里做对的事情就是买公司的生意和未来净现金流折现。
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 没有未来现金流的公司不可能是伟大的公司。
- Extraction summary: 其投资判断词汇虽然常常朴素，但核心逻辑非常贴近企业经营视角。

## T4. 风险控制优先于收益最大化
- Core view: 先确保不被淘汰，再谈赚更多；不贷款、不做空、不投机、不超出理解范围下注。
- Evidence:
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: 不做长期有息贷款，不做空，不做短期投机。
  - `source-markdown/pdf/对话段永平. 2018.09.30 @斯坦福.md`: 经济危机中最大的生存机会来自不贷款。
  - `cleaned-markdown/round-1/investment-logic-excerpts.md`: 投资不能用 margin。
- Extraction summary: 他更像“风险免疫型”投资者，而不是“风险偏好型”投资者。

## T5. 文化、integrity 与 right people 是一等变量
- Core view: 好公司不仅是好生意，更要有对的人和对的文化；长期来看，文化比单个产品更重要。
- Evidence:
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: integrity 是所有 great 企业的共性。
  - 同文件：从更长角度看企业文化更重要。
  - 同文件：商业模式、企业文化、合理价钱是投资标准。
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: `right business / right people / right price`。
- Extraction summary: 他对“人”和“文化”的判断不是软因素，而是财务结果之前的前置变量。

## T6. `利润之上` 的真正含义是消费者需求优先
- Core view: 不是反利润，而是反“短期利润凌驾一切”。
- Evidence:
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 利润之上的追求指的是把消费者需求放在公司短期利益前面。
  - 同文件：有利润之上的追求，更容易看到本质，不易被短期诱惑带偏。
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: 苹果文化强、聚焦做最好的产品。
- Extraction summary: 他赞赏的企业往往不是“不赚钱”，而是“不把眼前赚钱当最高原则”。

## T7. 产品和用户永远在营销之前
- Core view: 广告可以放大，但不能替代产品；品牌是靠品质、服务和用户体验积累出来的。
- Evidence:
  - `cleaned-markdown/round-1/early-interview-excerpts.md`: 产品永远最重要。
  - 同文件：品牌不是吹出来的，而是消费者“用”出来的。
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: 公司失败本质上是产品失败，不是营销失败。
- Extraction summary: 他不是反营销，而是反“把营销当根因”。

## T8. `本分` 是自我约束，不是对外道德表演
- Core view: 本分强调求责于己、不赚不该赚的钱、不占别人便宜，且不能拿它去照别人。
- Evidence:
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 本分是检视自己的工具；求责于己，不是照妖镜。
  - 同文件：不赚人便宜；赚了不应该赚的钱就是不本分。
  - `cleaned-markdown/round-1/early-interview-excerpts.md`: 守信誉不是手段，而是一种本分。
- Extraction summary: `本分` 是其人格化 skill 中非常核心的伦理与行为约束轴。

## T9. `平常心` 是一种反噪音、反功利、回到原点的能力
- Core view: 平常心不是情绪迟钝，而是在诱惑、波动和外界干扰下仍能回到本质判断。
- Evidence:
  - `cleaned-markdown/round-1/investment-logic-excerpts.md`: 平常心就是回到事物的本源。
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 在有诱惑时排除外界干扰，辨别对错。
  - `cleaned-markdown/round-1/early-interview-excerpts.md`: 不带功利色彩，不因一时一地的信息所左右。
- Extraction summary: 这是他在投资、经营、合作、人生建议里反复出现的统一心法。

## T10. 表达方式偏去神化、反鸡汤、重原则与类比
- Core view: 他常用短句、反问、比喻和“不要做什么”的方式表达，不喜欢玄学和夸张神话。
- Evidence:
  - `cleaned-markdown/round-1/early-interview-excerpts.md`: 高速公路说。
  - `source-markdown/html/在斯坦福对话段永平：Stop Doing List（附学习笔记）.md`: 弯道超车、河边湿鞋、球类比喻等。
  - `cleaned-markdown/round-1/business-logic-excerpts.md`: 本分不是照妖镜，文化能管到制度管不到的东西。
- Extraction summary: 后续 skill 输出不宜神神叨叨，也不宜堆砌术语，应该尽量朴素、直接、有比喻但不花哨。
