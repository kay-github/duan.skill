# duan-yongping-persona-skill

基于段永平公开投资问答与商业表达整理出的正式版人格化 skill 技能包。

当前状态：`formal v1.1`

## 主要文件
- `SKILL.md`
  - 正式版技能文件。
- `references/`
  - 核心支撑材料，包括来源覆盖、风格 DNA、反模式分析。

## 目录结构
- `SKILL.md`
  - 正式版 skill。
- `references/`
  - 正式版支撑文档。
- `source-markdown/`
  - 原始 PDF / HTML 转换后的 Markdown 来源层。
- `cleaned-markdown/`
  - 面向分析的清洗摘录层。
- `analysis/`
  - 问答对、主题块、来源说明等结构化分析层。
- `distilled/`
  - 人格维度、规则层等蒸馏结果。
- `archive/`
  - 过程性归档，包括计划、数据处理流程、评测用例和评测结果。
- `temp-ocr/`
  - OCR 中间产物，主要用于本地来源追溯和留档，默认不建议上传。

## 上传 GitHub 建议
- 如果要完整保留研究过程，直接上传整个技能包目录即可。
- 如果只想保留正式版与核心支撑，至少保留这些内容：
  - `SKILL.md`
  - `references/`
  - `source-markdown/`
- `archive/` 主要是过程记录，适合保留，但不是 skill 运行的最小必需集。

## 说明
- 本技能包复现的是“段永平式判断框架”，不是对真人的身份扮演。
- 仅基于当前资料库中的公开材料构建；若后续加入新资料，建议继续按现有层次追加到 `source-markdown/ -> cleaned-markdown/ -> analysis/ -> distilled/`。
